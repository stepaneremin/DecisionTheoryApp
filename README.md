# DecisionTheoryApp

Desktop-приложение для поддержки принятия решений, реализующее два метода многокритериальной оптимизации. Разработано в рамках учебного проекта на кафедре информационных систем СПбГЭТУ «ЛЭТИ».

## Методы

### Метод анализа иерархий (МАИ)
Классический метод Томаса Саати для выбора лучшей альтернативы по нескольким критериям через попарные сравнения.

- Добавление и переименование критериев и альтернатив
- Заполнение матриц попарных сравнений по шкале Саати (1–9)
- Автоматическое заполнение обратных значений
- Валидация ввода с блокировкой некорректных значений
- Расчёт весов критериев и итоговых приоритетов альтернатив
- Оценка согласованности матриц (ОС)
- **История расчётов** — сравнение результатов при изменении матриц

### Метод главного критерия
Метод многокритериальной оптимизации для дискретных и непрерывных областей.

- Ввод произвольных математических функций-критериев (до 3)
- Поддержка переменных (до 5): непрерывные `[min, max, шаг]` и дискретные `{точки}`
- Выбор главного критерия (максимум/минимум) и задание ограничений для остальных (≥, ≤, диапазон)
- Перебор всей области оптимизации, фильтрация допустимых точек
- Таблица результатов с подсветкой оптимума
- **История расчётов** — сравнение оптимальных точек при изменении параметров

## Синтаксис формул

| Запись | Смысл |
|--------|-------|
| `x^2` | x в квадрате |
| `2*x1 + 3*x2` | линейная функция |
| `Sqrt(x)` | квадратный корень |
| `Abs(x)` | модуль |
| `Sin(x)`, `Cos(x)` | тригонометрия (радианы) |
| `Exp(x)` | e^x |
| `Log(x)` | натуральный логарифм |
| `Log10(x)` | десятичный логарифм |

- Генерация отчётов в формате `.docx` (Word)
- Встроенная справка (HTML, открывается в браузере)

- **C# / .NET 8.0**

- **MVVM** — архитектурный паттерн
- **NCalc** — парсинг и вычисление математических выражений

## Запуск

1. Клонировать репозиторий
2. Открыть `DecisionTheoryApp.sln` в Visual Studio 2022+
3. Установить пакет `NCalc` через NuGet
4. Собрать и запустить (`F5`)

Для работы справки положить файлы `инструкция_МАИ.html` и `инструкция_метод_главного_критерия.html` рядом с `.exe`.
